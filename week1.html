Week 1 - Introduction to Embedded Systems

Week 1 - Introduction to Embedded Systems

Objectives

The primary objectives of Week 1 were to:

  • Introduction to HTML and CSS fundamentals
  • Comprehend the foundational concepts of embedded systems
  • Explore the architecture and constituent components of embedded systems
  • Installation and configuration of development environments (Thonny IDE and Arduino IDE)
  • Acquire hands-on experience programming embedded systems using MicroPython
  • Familiarize with microcontroller development boards, with emphasis on the Raspberry Pi Pico

Overview

The inaugural week of the embedded systems training program commenced with an exploration of web development fundamentals — a skillset that would prove indispensable for documenting our technical journey. I was tasked with constructing a simple HTML-based website upon which we would subsequently publish our weekly progress, experimental findings, and project deliverables.

HTML & Web Development Foundation

Through guided instruction, I acquired proficiency in the following domains:

  • HTML Structure: Understanding the document object model (DOM), semantic elements, headings, paragraphs, lists, links, and image embedding
  • CSS Styling: Mastering selectors, the box model, flexbox layout, color schemes, typography, and responsive design principles
  • File Organization: Proper directory structure for web projects
  • Hypertext Navigation: Creating inter-page links for seamless user experience

GitHub Repository & Deployment

Subsequently, I learned to deploy my creation to the world wide web through the following workflow:

  • Repository Initialization: Creating a Git repository to version-control our web project
  • Committing Changes: Staging and committing HTML, CSS, and asset files
  • Remote Synchronization: Pushing the codebase to GitHub
  • GitHub Pages Hosting: Enabling static site hosting directly from the repository
  • Live URL Generation: Obtaining a publicly accessible URL for the deployed website

You can view the repository here: Assignment Repository

Introduction to Embedded Systems

Parallel to web development, I was introduced to the fundamental concepts of embedded systems. I learned about the architecture of embedded systems, including the components such as microcontrollers, sensors, actuators, and communication interfaces. I also gained an understanding of the different types of embedded systems and their applications across various industries.

I explored the basics of programming embedded systems using MicroPython and acquired hands-on experience with microcontroller development boards, particularly the Raspberry Pi Pico.

Overall, the first week provided a solid foundation — both for web-based documentation and embedded systems fundamentals — setting the stage for more advanced topics in the ensuing weeks.

In the next week, I look forward to diving deeper into hardware and software integration, where I will learn how to interface sensors and actuators with microcontrollers and develop more complex embedded applications.

Key learning outcomes from Week 1:

  • Comprehension of embedded systems architecture and constituent components
  • Proficiency in HTML structure, CSS styling, and web project organization
  • Competence in GitHub repository creation, version control, and GitHub Pages deployment
  • Installation and configuration of integrated development environments
  • Foundational MicroPython programming for embedded platforms
  • Practical experience with the Raspberry Pi Pico microcontroller

Week 1 proved to be an exhilarating commencement to the embedded systems training program. The dual focus on web development and embedded fundamentals has established a robust platform upon which more advanced topics shall be built.

In the subsequent week, I anticipate delving into hardware-software integration — learning to interface sensors and actuators with microcontrollers while developing increasingly sophisticated embedded applications.
