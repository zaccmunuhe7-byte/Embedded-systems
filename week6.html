Week 6 - Introduction to Raspberry Pi Pico W

Overview

This week, we dived into the world of microcontrollers with a focus on the Raspberry Pi Pico W. The Pico W is a powerful and versatile microcontroller board that combines the capabilities of the original Raspberry Pi Pico with built-in Wi-Fi connectivity, making it an excellent choice for IoT projects and embedded systems development.

We were able to configure it to run MicroPython, a popular programming language for microcontrollers, which allows for rapid development and easy prototyping. We explored the various features of the Pico W, including its GPIO pins, ADC capabilities, and Wi-Fi functionality.

Through hands-on exercises, we learned how to write MicroPython code to control the Pico W, read sensor data, and establish wireless communication. This week's training provided a solid foundation for working with the Raspberry Pi Pico W and set the stage for more advanced projects in the coming weeks.

Hands-on Projects

We were able to integrate the Pico with ultrasonic sensors and I2C LCD displays and buzzer to measure distance and display information.

Project Demonstrations

We also had the opportunity to light the inbuilt LED and external RGBs at specified intervals.

Key Learning Outcomes

  • Understanding the features and capabilities of the Raspberry Pi Pico W
  • Configuring the Pico W to run MicroPython using Thonny
  • Writing MicroPython code to control the Pico W and read sensor data
  • Lighting the inbuilt LED and external RGBs at specified intervals
  • Establishing wireless communication using the Pico W’s Wi-Fi functionality
  • Gaining hands-on experience with a powerful microcontroller platform for IoT and embedded systems development

Overall, Week 6 was an exciting introduction to the Raspberry Pi Pico W, and I look forward to applying these skills in more advanced IoT projects.
